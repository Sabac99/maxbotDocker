# 1. Установка зависимостей

## Установить зависимости из requierments.txt

### В requierments.txt описаны все использовавшиеся пакеты и библиотеки, в то же время для минимального запуска без разработки необходимо проверить наличие пакетов в "packages"

# 2. Запуск контейнера
### Сначала необходимо собрать образ из Dockerfile:
```bash
docker build -t maxbot-timie .
```

### Затем необходимо выполнить следущий скрипт
```bash
bash ./start_bot.sh
```
